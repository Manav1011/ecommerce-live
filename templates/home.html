Core Functionalities
Fully responsive and runnable on any device
Both registered and non-registered users can make order
When user creates an account an activation link is sent to his/her email address. user can activate their account by clicking on the activation link which is a one time link and valid for 5 minutes
Only users having activated accounts are able to login
At the time of login django authentication system authenticates the user and only then they are allowed to login
User can also create guest accounts and use full functionalities of the site
User can search for the products by their title, tags, descriptions, price.
Ajax has been used to enable asynchronous functionalities like search predictions, adding or removing product from the cart and many more.
User can reset his forgotten password and can also change the existing password
User can create a order and pay easily with any of the available payment options.

Technology Used
Backend Language - Python
Framework - Django 4.0.5
Frontend Language - HTML 5, CSS 3
Scripting Language - Javascript ES2015
Additional Libraries - Bootstrap, Jquery
AJAX for asynchronous communication
